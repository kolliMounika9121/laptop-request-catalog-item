📌 Project Overview

The Laptop Request Catalog Item project is developed using the ServiceNow platform to streamline and automate the internal laptop request process within an organization. The solution is designed to simplify how employees request laptops, how managers approve requests, and how IT administrators fulfill and track those requests efficiently.

This project leverages ServiceNow’s Service Catalog, UI Policies, UI Actions, Approval Workflows, Notifications, and Update Sets to create a structured, secure, and transparent request management system.

🎯 Purpose of the Project

In many organizations, laptop allocation is handled manually through emails or informal communication, leading to delays, lack of tracking, and limited accountability.

This project aims to:

Automate the laptop request process

Ensure proper managerial approval

Enable IT administrators to track and fulfill requests

Provide real-time request status visibility

Maintain auditability and governance

⚙️ Key Features

Role-Based Access (Employee, Manager, IT Admin)

Laptop Request Form via Service Catalog

Dynamic Form Behavior using UI Policies

Reset Button using UI Action

Manager Approval Workflow

IT Admin Fulfillment Process

Email Notifications for Status Updates

Request Tracking via sc_request and sc_req_item tables

Update Set Export/Import for Cross-Instance Deployment

🏗️ Platform & Version

Platform: ServiceNow

Version: Zurich

Module Used: Service Catalog

🚀 Expected Outcome

The system ensures transparency, accountability, and efficiency in laptop allocation by automating approvals, enforcing validation rules, and maintaining a structured request lifecycle from submission to fulfillment.
